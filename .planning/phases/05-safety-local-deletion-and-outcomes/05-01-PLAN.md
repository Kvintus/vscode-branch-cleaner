## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| User selection vs Git | User explicitly selects branches in QuickPick; code must not widen scope beyond that set. |
| Local vs remote | Extension must not invoke remote deletion or push that drops remote refs. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-05-01 | Tampering | Misleading confirm copy | mitigate | Baseline string equals `plan.baseline.displayLabel`; risky branches listed with merge states matching `mergeDetailLine` semantics. |
| T-05-02 | Elevation | Force-delete without confirm | mitigate | `deleteBranch(name, true)` only for `not_merged`/`unknown` after modal confirm; `merged` uses `force: false`. |
| T-05-03 | Denial | Silent partial failure | mitigate | Modal lists every branch; failures first (**SAFE-03**). |

Secrets / command injection: **N/A** (branch names from Git API; no shell).
</threat_model>

    Use **`import type { Repository } from '../types/git'`** (adjust relative path if needed).
